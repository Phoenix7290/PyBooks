# TP 2

Neste TP, seu professor irá disponibilizar uma base de dados simulada, contendo informações sobre os funcionários de uma empresa fictícia.
A tabela conterá campos como ID do funcionário, nome, cargo, departamento, salário, data de contratação, entre outros atributos relevantes. O professor irá pedir 15 tarefas baseadas em consultas SQL.

A avaliação deste item do projeto será realizada através da execução das sequências de SQL que atendam aos requisitos de cada etapa. Serão considerados critérios como precisão na extração de dados, uso correto dos operadores, clareza na escrita das consultas e a capacidade de interpretar corretamente o que está sendo pedido.

Este projeto integrado oferece uma oportunidade para você desenvolver habilidades práticas em consultas SQL, permitindo-lhes explorar e analisar dados de funcionários de uma empresa, aplicando os conceitos aprendidos sobre operadores de comparação, lógicos, ordenação e agregação de dados.

Nota especial: O projeto pode ser adaptado para o seu. Mas mantenha como se fosse um Login. Podendo ser login de Funcionário ou Cliente.
