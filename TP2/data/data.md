# PyBooks

Arquivos incluídos:

- create_db.sql: Criação do banco e tabela
- insert_data.sql: Inserção dos 20 registros
- queries.sql: 15 consultas SQL pedidas
- pybooks_db_dump.sql: Dump completo do banco
- queries_results.txt: Resultados das consultas

Instruções:

1. Importe pybooks_db_dump.sql com: mysql -u root -p < pybooks_db_dump.sql
2. Rode as consultas com: mysql -u root -p pybooks_db < queries.sql
