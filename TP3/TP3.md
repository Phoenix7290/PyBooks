# Orientações para o TP3 - 2025

Com base nos encontros de Projeto de Bloco e alinhado às diretivas do TP3 do Projeto de Bloco, você deve:

1 - Criar o Modelo Lógico do seu Projeto (use qualquer modelador)
2 - Crie o script DDL a partir do seu modelo para criar o BD em PostgreSQL com todos os objetos pertinentes (tabelas, relacionamentos e etc.)
3 - Popule as tabelas com dados básicos usando scripts DML
4 - Realize um script DQL contendo três consultas, a saber:
   a) consulta com INNER JOIN
   b) consulta com LEFT JOIN
   c) consulta com RIGHT JOIN
  
5 - Crie um código em Python, usando SQLAlchemy, para conectar a esta base de dados e consumir cada uma dessas consultas, populando dicionários.

6 - Faça um loop imprimindo na tela o conteúdo dos dicionários, para cada consulta.

7 - Refaça a questão #5, porém, dessa vez, populando listas

8 - Faça um loop imprimindo na tela o conteúdo das listas, para cada consulta.
