# TP 5

Neste item de projeto, você irá exercitar técnicas de web crawling, web scraping e manipulação de dados em SQL. Sob a orientação do professor, você irá explorar a recuperação de informações de páginas HTML, o armazenamento dos dados extraídos em tabelas SQL e a aplicação de operações de junção para análise e extração de insights dos dados coletados.

Atividades:

Web Crawling e Web Scraping: Você irá baixar e analisar o conteúdo HTML de páginas web especificadas pelo professor. Utilizando bibliotecas como urllib e BeautifulSoup em Python, você extrairá informações específicas destas páginas, navegando pela estrutura de atributos do HTML e aplicando expressões regulares conforme necessário.
Armazenamento em Tabelas SQL: Os dados extraídos serão armazenados em tabelas SQL, com a estrutura definida pelo professor. Além disso, os metadados de cada página utilizada para scraping serão registrados em outra tabela SQL.
Manipulação de Dados em SQL: Você realizará consultas SQL que envolvam operações inner join, permitindo a manipulação de informações provenientes de múltiplas tabelas.
Gestão de Exceções e Relatório Final: Durante o processo de scraping, você lidará com exceções para tratar possíveis erros, registrando informações sobre essas ocorrências em uma terceira tabela SQL. Ao final, você produzirá um relatório que apresentará o conteúdo extraído, a quantidade de falhas por site visitado e outras métricas determinadas pelo professor.
Este item de projeto proporciona uma oportunidade valiosa para você adquirir e aplicar habilidades avançadas em web scraping, manipulação de dados em SQL e gestão de exceções.
